 # ML Classification Metrics

A Python project to **evaluate machine learning classification models** using common metrics like **accuracy, precision, recall, F1 score, and false positive rate (FPR)**. Implemented using **Scikit-learn**.

---

## Features

- Compute confusion matrix
- Calculate:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - False Positive Rate (FPR)
- Visualize ROC curve and thresholds
- Easy to adapt to your own dataset

---
